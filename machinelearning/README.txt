Данное программное обеспечение предназначено для обработки и исправления текстов на ладино, написанных шрифтом Раши. Для корректной работы программы необходимо использовать Python 2.7 с установленным модулем scikit-learn.

Чтобы обработать текст, необходимо:
1) скачать программу the_fixer.py и файлы, названия которых кончаются на .pkl и npy.
2) положить все файлы в одну папку.
3) положить в ту же папку текст для обработки.
4) запустить программу и ввести имя файла, когда программа предложит.
